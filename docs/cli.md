# Command-Line Interface (CLI) Documentation

## Overview

The `fm` CLI provides command-line access to the factory library's factor modeling capabilities.

## Installation

Build the project and the CLI executable will be available at `build/src/fm`.

## Basic Usage

```bash
# Display help
./build/src/fm --help

# Run with input data
./build/src/fm --input data.csv --output results.json
```
